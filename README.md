# 📈 Forex Trading Signal AI Agent

An end-to-end **AI-powered Forex Trading Signal System** built using **Python, Machine Learning, Technical Analysis, Backtesting, DAX, and Power BI**.

This project analyzes historical Forex market data, generates AI-driven BUY/SELL/HOLD trading signals, evaluates trading strategies through backtesting, and presents actionable insights using interactive Power BI dashboards.

---

## 🚀 Project Overview

The project combines technical indicators, machine learning, and business intelligence to simulate an AI-assisted Forex trading platform.

### Key Features

- 📊 Forex market data analysis
- 📈 Technical indicator calculation
- 🤖 AI-based trading signal generation
- 💹 Strategy backtesting
- ⚠️ Risk analysis
- 📉 Machine Learning predictions
- 📋 Interactive Power BI dashboards

# 📊 Technical Indicators

The following indicators were implemented:

- Simple Moving Average (SMA 10)
- Simple Moving Average (SMA 50)
- Relative Strength Index (RSI)
- MACD
- Bollinger Bands
- Average True Range (ATR)

These indicators are combined to generate BUY, SELL, HOLD, STRONG BUY, and STRONG SELL trading signals.

# 📈 Backtesting

The generated trading signals are evaluated using a backtesting engine.

Performance metrics include:

- Total Net Profit & Loss
- Cumulative P&L
- Win Rate
- Maximum Drawdown
- Strategy Comparison
- Filtered Strategy Performance
